ระบบนี้ช่วยดูแลการปลูกพริกขี้หนู (Chilli Padi) ให้เติบโตได้ดีขึ้นโดยการ ควบคุมสภาพดินให้อยู่ในเกณฑ์ที่เหมาะสมที่สุดสำหรับการเจริญเติบโต ผ่านการทำงานอัตโนมัติ 3 ส่วนหลัก ได้แก่ การตรวจสอบความชื้น การวัดค่าความเป็นกรด-ด่าง (pH) และการเติมปุ๋ยเมื่อจำเป็น
ระบบนี้ช่วยเพิ่มประสิทธิภาพการปลูกพริกขี้หนูผ่านกระบวนการต่างๆ ดังนี้:
1. การควบคุมความชื้นในดินให้เหมาะสม (Soil Moisture Monitoring)
พริกขี้หนูเป็นพืชที่ต้องการดินที่มีความชื้นและต้องรดน้ำอย่างสม่ำเสมอ ระบบนี้จึงช่วยป้องกันปัญหาน้ำน้อยหรือมากเกินไปโดย:
• กำหนดค่าความชื้นที่เหมาะสม: ระบบตั้งค่าความชื้นเป้าหมายสำหรับพริกขี้หนูไว้ที่ 50% – 70%
• การแจ้งเตือน: เซนเซอร์จะวัดระดับความชื้นตลอดเวลา
    ◦ หากความชื้น ต่ำกว่า 50%: ระบบจะแจ้งเตือนให้ "เติมน้ำ" (Add Water) และไฟ LED สีแดงจะติด
    ◦ หากความชื้น สูงกว่า 70%: หรืออยู่ในเกณฑ์ที่เหมาะสม ระบบจะแจ้งให้ "หยุดเติมน้ำ" (Stop Add Water) และไฟ LED สีเขียวจะติด เพื่อป้องกันรากเน่าจากน้ำขัง
2. การควบคุมค่า pH เพื่อการดูดซึมสารอาหาร (Soil pH Monitoring)
ค่า pH ของดินมีผลโดยตรงต่อความสามารถของพืชในการดูดซึมสารอาหาร ระบบนี้ช่วยรักษาค่า pH ให้อยู่ในเกณฑ์ที่พริกขี้หนูเติบโตได้ดีที่สุดคือ 5.5 ถึง 6.5 โดยมีกลไกดังนี้:
• การตรวจสอบแบบ Real-time: ใช้หัววัด pH Meter วัดค่าความเป็นกรด-ด่างตั้งแต่ 0-14
• การแจ้งเตือนความผิดปกติ:
    ◦ pH ต่ำกว่า 5.5 (เป็นกรดเกินไป): ไฟ LED สีแดงจะติดเพื่อเตือน
    ◦ pH สูงกว่า 6.5 (เป็นด่างเกินไป): ไฟ LED สีแดงจะติดเพื่อเตือน
    ◦ pH ปกติ (5.5 - 6.5): ไฟแจ้งเตือนจะดับลง แสดงว่าดินอยู่ในสภาพสมบูรณ์
3. การเติมปุ๋ยอัตโนมัติ (Automated Fertilizer Addition)
ระบบมีการติดตั้งมอเตอร์เซอร์โว (Servo Motor) ที่ทำหน้าที่เป็นแขนกลสำหรับเติมปุ๋ย ซึ่งเชื่อมโยงกับการวิเคราะห์ค่า pH:
• การทำงานสัมพันธ์กับค่า pH: หากระบบตรวจพบว่าค่า pH สูงกว่า 6.5 (ดินเป็นด่าง) ระบบจะส่งสัญญาณให้มอเตอร์ทำงานเพื่อตักหรือปล่อยปุ๋ยลงในดินโดยอัตโนมัติ
• ระบบหน่วงเวลา: หลังจากเติมปุ๋ย ระบบจะรอประมาณ 10 วินาทีเพื่อให้ค่า pH คงที่ก่อนจะวัดผลซ้ำ หากค่า pH ยังสูงอยู่ กระบวนการเติมปุ๋ยจะทำงานอีกครั้ง
4. การติดตามผลระยะไกล (Remote Monitoring via App)
ระบบใช้อุปกรณ์ WiFi (ESP32) เชื่อมต่อกับแอปพลิเคชัน Blynk บนมือถือ ช่วยให้เกษตรกรดูแลสวนพริกได้สะดวกยิ่งขึ้น:
• ข้อมูลแบบ Real-time: แสดงค่าความชื้นและ pH ปัจจุบันบนหน้าจอมือถือ
• การแจ้งเตือนทันที: หากค่าต่างๆ ผิดปกติ แอปฯ จะแจ้งเตือนให้ผู้ปลูกทราบทันที เพื่อให้เข้าไปแก้ไขได้ทันท่วงที
• การตัดสินใจแม่นยำ: ช่วยให้การจัดการน้ำและสารเคมีเป็นไปอย่างแม่นยำ ประหยัดทรัพยากร และช่วยให้พืชแข็งแรงขึ้นในระยะยาว
สรุปคือ ระบบนี้ช่วยลดความเสี่ยงในการปลูกพริกขี้หนูโดยใช้เทคโนโลยีเข้ามาช่วยตัดสินใจและทำงานแทนมนุษย์ในจุดที่ต้องการความแม่นยำ เพื่อให้มั่นใจว่าพริกจะได้รับน้ำและสารอาหารในปริมาณที่ถูกต้องที่สุด
